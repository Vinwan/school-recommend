# 高校推荐系统

这是一个基于Flask的高校推荐系统，可以根据用户输入的分数推荐合适的高校。

## 功能特点

- 根据分数智能推荐高校
- 支持分数范围自动调整
- 显示学校详细信息（包括省份、专业组、投档线等）
- 使用SQLite数据库存储学校数据
- 支持CSV数据导入

## 技术栈

- 后端：Python + Flask
- 数据库：SQLite
- 前端：HTML + CSS + JavaScript
- 数据处理：Pandas
